6パターンスコアリング

「本当にコンセプトが刺さって売れている」クリエイティブを検出する6つの仮説

Pattern A: 爆発力

cost_difference / streaming_period_days
短期爆発 = 市場が即座に認めた

Pattern B: 持続力

streaming_period × (cost_diff / top20_avg)
長期配信 = CPAが安定的に合っている

Pattern C: 拡散力

digg_count_diff / play_count_diff
見た人のアクション率（video専用）

Pattern D: 横展開力

active_platforms × cross_cost_ratio
複数PF展開 = 勝ちCRの証拠

Pattern E: コンセプト効率

item_cost_diff / product_max_cost_diff
商材内A/Bテストの勝者

Pattern F: 複合スコア

A〜E の加重平均。banner用/video用で重み分離。

実験から得られた構造的な学び

Pattern A/B分離がクラスタレベルで機能

爆発力TOP50と持続力TOP50の重複率はわずか36%。2つの軸は独立した情報を持ち、別々に分析する価値がある。

最強コンセプト構造の条件

「成分名訴求 × UGCレビュー × 具体悩み」構造は爆発力と持続力の両方を持つ。この構造は他ジャンルに転用可能。

ベクトル空間のスイートスポット

cos_sim 0.7〜0.85 = 刺さる＆差別化あり
cos_sim 0.85〜1.0 = レッドオーシャン
cos_sim < 0.5 = 市場性不明

バリュエーションデータ = コアコンピタンス

「仮説 × スコアリング × 結果」の三つ組はループを回さないと手に入らない。データの量ではなく「判断の累積」が競争優位。
